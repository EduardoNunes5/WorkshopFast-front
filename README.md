# WorkshopfastFront

O Workshop fast é uma aplicação frontend capaz de listar os detalhes dos colaboradores e workshops da Fast!

Funcionalidades:
- listagem de workshops
- detalhes de um workshop
- listagem de detalhes do workshop, como os colaboradores que participaram

O projeto foi feito utilizando Angular, [Angular CLI](https://github.com/angular/angular-cli) versão 19.2.19.

## Executando o projeto:

* Tenha node versão 18.20.x ou superior na maquina
* Instale as dependências com npm install
* execute `ng serve`


Caso o seu backend esteja executando em uma porta diferente da 8080, alterar a porta da baseUrl no arquivo `src/environments/environment.development.ts`
